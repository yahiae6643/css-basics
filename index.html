CSS Basics

Before You Begin

There are two files:

  • index.html is the content. DO NOT EDIT ANY OF IT.
  • style.css is the style. This is where you will be doing your coding.

Follow the directions on this HTML page below.

The Wrapper

You'll notice that <div id="wrapper"> has already been styled for you. Don't worry too much about the declarations inside that rule, but feel free to turn each declaration off/on to see what they do.

The Horizontal Rules

Let's start simple. Just copy and paste the code snippet below to style all of the <hr> elements.

hr {
    border: 0;
    height: 1px;
    background: #333;
    background-image: linear-gradient(to right, #ccc, #333, #ccc);
}

Subtle, but awesome. Right?

The Headings

Select all <h1> elements.

  • Set the font-size to 72px

In a new rule, select all <h1> and <h2> elements at the same time.

  • Set the font-family to "Oswald", "Helvetica Neue"
    • This means that the font will be Oswald (an imported font, more on that later), and Helvetica Neue is the closest backup in case Oswald doesn't load.
    • Fonts only need to be surrounded with quotes if they are multiple words, such as "Times New Roman", but it's a good habit to surround all fonts with quotes, just in case.
  • Set the text-align property to center
  • Set the color to #2884C3

Starting to look familiar?

Text

Remember that everything in <HTML> is either in the <head> or <body>

We can select all text by selecting the <body>

  • and setting the font-size to 1.5em
    • em means emphasis and simply multiplies the current font size by your # (in this case, 1.5).

This declaration just applied to everything in the body that has a font-size (namely, text - such as p, li, h2, etc.)

But why aren't <h1> elements affected? Remember that rules cascade, meaning that h1 is more specific than body, so h1 declarations overrule any body declarations.

I'm not a big fan of the font, so let's change the font family, too. Similar to the <h1> and <h2> elements, one has already been imported for you.

  • set the font-family to "Crimson Text", "Arial"

But now the text seems to squished together, right? We can space out each line by

  • setting the line-height to 1.5

Code

Select all <code> elements.

  • Set the color to #CE234C

Did you know you can use words, too? Try it!

  • Set the background-color to lightgray

By doing this, hopefully you noticed that everything looks good, except for that multi-line code snippet at the top! How can we make the entire background of that code gray? We'd have to put it in a container (think: div) and make the entire container have a gray background. The HTML part has been done for you. And in case we want to do a code snippet again, the div has class="snippet"

So select all elements with that class.

  • Set the background to lightgray
